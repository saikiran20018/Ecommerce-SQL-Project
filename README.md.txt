# 🛒 E-Commerce Database (MySQL Project)

## 📌 Project Overview
This project is a fully designed relational E-Commerce database developed using MySQL Command Line. 
It simulates a real-world online shopping system similar to Amazon or Flipkart.

The database includes proper normalization, primary and foreign key relationships, constraints, and analytical business queries.

---

## 🗄 Database Schema

The database contains 6 main tables:

- Users
- Categories
- Products
- Orders
- Order_Items
- Payments

Each table is properly linked using foreign key constraints to maintain data integrity.

---

## 🔥 Features Implemented

- Relational Database Design
- Primary & Foreign Keys
- Data Normalization
- Sample Data Insertion
- Business Analytical Queries
- Revenue & Sales Analysis
- Indexing Support

---

## 📊 Example Business Queries

- Total Revenue Calculation
- Top Selling Products
- Monthly Revenue Report
